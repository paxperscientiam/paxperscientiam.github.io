The provided image ﬁles can be included as normal using i.e. using the

options
i]{h

The
mwe

graphicx
,
lipsum

blindtext
,

PDF images. Also
\graphicspath{{example-}}

Some MWE might even be better oﬀ not to use the package if speciﬁc
side-eﬀect between packages is tested.
